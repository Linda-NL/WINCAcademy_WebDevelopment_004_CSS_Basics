oefening #4: CSS basics

Mijn eerste HTML pagina gestyled in CSS. Ik heb in head een link gemaakt naar het .css bestand. Ik heb alle pagina's vormgegeven d.m.v. css.
Ik ben benieuwd hoe ik een fatsoenlijke navigation bar maak. En hoe ik dingen naast elkaar kan gebruiken. En tekst uitlijnen is ook nog niet gelukt met de css selectors die tot nu toe behandeld zijn. Ik gok dat er iets van een box nodig is, waarbinnen je margins kunt geven aan je tekst.

"Hoe zorg je er voor dat je css stylesheet geen rommeltje wordt?"

Ik vind de html rommelig worden als je een style in een style wil toepassen en daarvoor de div-tag gebruikt: div-inception. Ik wil graag leren hoe dit netter kan.
Ik heb de article-tag gebruikt voor css-vormgeving, maar ik kom er nu achter dat dat niet handig is. Want ik wil hier dezelfde opmaak gebruiken, maar dit is natuurlijk geen op zichzelf staand artikel, dus moet ik hier de article-tag niet gebruiken.
